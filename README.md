#### Documentation

Api documentation is available on the [API website](https://api.akeneo.com/api-reference.html)</a>

#### Available api

- **Product**
- **Product model**
- **Media files**
- **Family**
- **Family variants**
- **Attribute**
- **Attribute options**
- **Attribute groups**
- **Association type**
- **Category** 
- **Channel** 
- **Locale** 
- **Currency** 
- **Measure family**
